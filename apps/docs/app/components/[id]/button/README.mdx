# Button

`Button` 组件主要服务于用户交互中 `点击` 的场景。使用该组件往往说明这个元素可被用户点击，进行一些交互反馈，例如 表单提交，路由跳转 等。

<Demo />

## Usage

```tsx
import { Button } from "@shilong/rc/components";

function App() {
  return (
    <>
      <Button>nihao</Button>
    </>
  );
}
```
